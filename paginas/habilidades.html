Habilidades

Adobe Xd

Es una intuitiva y potente herramienta de diseño de UI y UX para diseñar, crear prototipos y compartir la experiencia del usuario y diseño de elementos interativos para entornos digitales.
Tiene todo lo que necesitas para diseñar sitios web, aplicaciones, interfaces por voz y juegos.

Illustrator

Software de gráfico vectoriaes que te permite crear de todo, desde gráficos para web y dispositivos móviles hasta logotipos, iconos e ilustraciones.
Nítido en cualquier tamaño esta aplicación te permite crear gráficos o impresiones digitales en cualquier tamaño desde pegatinas hasta vallas publicitarias.

Photoshop

Desde la edición y composición fotográfica hasta la pintura digital, la animación y el diseño gráfico se usa photoshop.
Recorta, reemplaza, mezcla y destaca, carteles, embalajes, banners, sitios web. Combina fotos, gráficos y textos para crear imagenes y totalmente nuevas.

HTML5

Es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos.

CSS

El CSS, en español «hojas de estilo en cascada», es un lenguaje de diseño gráfico, utilizado para definir y crear la presentación de un documento estructurado, escrito en un lenguaje de marcado.
El CSS permite controlar la apariencia de una página web.
